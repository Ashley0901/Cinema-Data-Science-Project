#  Movie Industry Data Science Suite: Prediction & Recommendation

This project provides a comprehensive end-to-end data science workflow applied to the movie industry. Using two extensive datasets, it covers everything from data cleaning and exploration to the deployment of three distinct Machine Learning models.

## Data sets used for this project:
https://drive.google.com/drive/u/0/folders/1_dfhx5HXQRAbl5VRjBCdZ5x4lH-ExgoP
## Key Features

The project is structured into four main phases:

1.  **Exploratory Data Analysis (EDA):** Deep dive into movie trends, budget distributions, and revenue correlations.
2.  **Recommendation System:** A similarity-based engine that suggests movies based on a user's input (e.g., suggesting *Finding Dory* if you liked *Finding Nemo*).
3.  **Revenue Prediction (Regression):** A model designed to estimate a film's potential box office earnings based on its features.
4.  **Success Classification:** A classifier that categorizes movies as "Hits" or "Flops" based on commercial performance metrics.

## Tech Stack
* **Language:** Python 3
* **Libraries:** Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn.
* **Environment:** Jupyter Notebook / Google Colab.
